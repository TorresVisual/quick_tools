# Quick Tools

A minimal Windows system tray launcher for scripts, shortcuts, executables, folders, and URLs. Left-click the tray icon to open a dark popup with all your tools as clickable cards. Config reloads on every open — no restart needed after editing.

---

## Usage

**Run from source**
```
python quick_tools.py
```

**Build a standalone `.exe`**
```
python build.py
```
Output lands in `dist/quick_tools.exe` with a copy of `config.json` alongside it. A Start Menu shortcut is created automatically.

**Add / remove tools interactively**
```
python add_tool.py
```

---

## Tray behavior

| Action | Result |
|---|---|
| Left-click icon | Toggle the tools popup |
| Right-click icon | Menu: Open, Edit Config, Exit |
| Click a tool card | Launch the tool, close popup |
| Click outside popup | Close popup |
| `Escape` | Close popup |
| "Edit Config" button | Open `config.json` in Antigravity |

## config.json

`config.json` lives next to the executable (or next to `quick_tools.py` when running from source). It is reloaded from disk every time the popup opens, so changes take effect immediately.

### Full schema

```json
{
  "tools": [
    {
      "name":     "My Tool",
      "command":  "D:\\Scripts\\run.bat",
      "type":     "bat",
      "color":    "#f59e0b",
      "category": "Scripts",
      "args":     ["--flag", "value"]
    }
  ]
}
```

| Field | Required | Description |
|---|---|---|
| `name` | yes | Label shown on the card |
| `command` | yes | Path, bare exe name, or `https://` URL. Supports `%ENV_VAR%` expansion. |
| `type` | no | Override auto-detected type (see table below) |
| `color` | no | Hex accent color for the card. Defaults to type color if omitted. |
| `category` | no | Groups cards under a section header. Cards without a category appear ungrouped at the top. |
| `args` | no | List of extra arguments passed to the command |

### Supported types

Type is auto-detected from the file extension or URL prefix. Use `"type"` to override.

| Type | Detected from | Launch method | Default color |
|---|---|---|---|
| `bat` / `cmd` | `.bat`, `.cmd` | `cmd /k` in new console window | `#f59e0b` amber |
| `ps1` | `.ps1` | PowerShell, `-ExecutionPolicy Bypass`, new console | `#3b82f6` blue |
| `exe` | `.exe` or no extension | `ShellExecuteW` (handles UAC) | `#10b981` green |
| `py` | `.py` | Python interpreter, new console | `#eab308` yellow |
| `lnk` | `.lnk` | Shell `start` | `#8b5cf6` violet |
| `url` | `http://`, `https://` | Default browser | `#6366f1` indigo |
| `folder` | explicit `"type": "folder"` | Windows Explorer | `#f97316` orange |

### Working directory

All tools with an absolute path in `command` are launched with their working directory set to the file's parent folder. This ensures scripts that reference relative paths or sideload DLLs work correctly.

### Examples

```json
{ "name": "Backup",      "command": "D:\\Scripts\\backup.bat",              "category": "Scripts" }
{ "name": "SSH Dev",     "command": "D:\\Scripts\\ssh_dev.ps1",             "category": "Scripts" }
{ "name": "My App",      "command": "D:\\Apps\\MyApp\\MyApp.exe",           "category": "Tools"   }
{ "name": "Downloads",   "command": "%USERPROFILE%\\Downloads", "type": "folder", "category": "Folders" }
{ "name": "GitHub",      "command": "https://github.com",                   "category": "Web"     }
{ "name": "Run Tests",   "command": "D:\\Project\\test.py", "args": ["-v"], "category": "Dev"     }
```

## add_tool.py

Interactive terminal editor. Runs in a loop — reloads config before each action.

```
python add_tool.py
```

- **A** — Add a tool. Prompts for name, command, optional type override, category (shows existing ones), color (defaults to type color), and args.
- **D** — Delete a tool by its list number.
- **Q** — Quit.

## File structure

```
quick_tools/
├── quick_tools.py     Main app — tray, popup, launcher
├── add_tool.py        Interactive terminal config editor
├── config.json        Your tools list (edit freely)
├── create_icon.py     Generates quick_tools.ico (run before first build)
├── build.py           PyInstaller build script → dist/
└── quick_tools.ico    Tray/exe icon (generated by create_icon.py)

dist/                  Created by build.py (git-ignored)
├── quick_tools.exe
└── config.json
```

## Dependencies

```
pip install pystray pillow pywin32
```

`pywin32` is only needed for `build.py` (creates the Start Menu shortcut). The app itself only needs `pystray` and `pillow`.
